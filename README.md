# 🤖 AI Personal Assistant (Concurrent Agent Edition)

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![OpenAI](https://img.shields.io/badge/LLM-OpenAI_gpt--4o--mini-green.svg)](https://openai.com/)

Виконане тестове завдання на позицію **Python Developer — AI Agents**.

Проєкт являє собою повністю асинхронного AI-агента, який виконує роль розумного особистого помічника. Агент автономно визначає наміри користувача (Intents), паралельно викликає необхідні інструменти (Function Calling) та підтримує безперервний контекстний діалог у консолі.

## 🎯 Реалізований функціонал
Окрім базових вимог ТЗ (погода та курс валют), архітектуру було розширено для демонстрації навичок побудови масштабованих систем, роботи з різними типами API та взаємодії з ОС.

* **🧠 Розумне розпізнавання намірів:** Агент використовує `OpenAI Tool Calling` для маршрутизації запитів.
* **⚡ Конкурентне виконання (Concurrency):** Завдяки `asyncio.gather`, агент здатний викликати декілька інструментів одночасно (наприклад, перевірити погоду і курс валют за один запит).
* **🗂 Менеджмент пам'яті:** Реалізовано механізм `_trim_history`, який запобігає переповненню контекстного вікна під час довгих сесій.
* **🎨 Інтерактивний UI:** Використано бібліотеку `rich` для красивого логування, спінерів завантаження та форматування відповідей у Markdown.

## 🛠 Арсенал інструментів (Tools)
Усі інструменти винесені в окремий ізольований модуль `tools/` для зручного масштабування:

1. 🌤 **Погода (`Open-Meteo`):** Детальний прогноз (температура, вологість, вітер, "відчувається як") із попереднім геокодуванням міста.
2. 💵 **Фіатні валюти (`ExchangeRate-API`):** Актуальні курси конвертації між 160+ світовими валютами.
3. 🪙 **Криптовалюта (`Binance API`):** Миттєвий парсинг поточної ціни будь-якої криптомонети у парі з USDT.
4. 📊 **Фондовий ринок (`Finnhub`):** Аналітика акцій (поточна ціна, ціна відкриття, максимуми/мінімуми дня).
5. 📰 **Новини (`NewsAPI`):** Пошук останніх світових новин за ключовим словом із генерацією клікабельних посилань.
6. 📚 **Вікіпедія (`Action API`):** Розумний парсер енциклопедичних довідок з обходом WAF-блокувань (коректні User-Agent та URL-кодування).
7. ⏱ **Світовий час (`TimeAPI.io`):** Точний локальний час за часовими поясами (IANA-формат).
8. 💻 **Системний сканер (`psutil` / `WMI` / `sysctl`):** Кросплатформний інструмент взаємодії з ОС. Аналізує залізо (CPU, GPU, RAM, Диски, Батарею, Температуру) як на **Windows**, так і на **macOS**.
9. 🔳 **Генератор QR-кодів (`qrcode`):** Створення графічних файлів (`.png`) з QR-кодом і збереження їх на локальний диск.
10. 🔗 **Скорочувач посилань (`TinyURL`):** Генерація коротких посилань із довгих URL.

## ⚙️ Структура проєкту
```text
my_agent/
├── core/
│   ├── config.py         # типізоване завантаження змінних середовища
│   └── logger.py         # налаштування rich-логгера
├── tools/                # ізольовані модулі інструментів
├── agent.py              # ядро: логіка агента, управління пам'яттю та виклик LLM
├── main.py               # точка входу (інтерактивний консольний цикл)
├── .env.example          # шаблон ключів API
├── requirements.txt      # залежності проєкту
└── README.md

🚀 Детальна інструкція із запуску
Для запуску проєкту необхідний Python версії 3.10 або вище.

Крок 1. Клонування репозиторію
Відкрийте термінал та виконайте команду:

Bash
git clone https://github.com/Bard368/Test
cd my_agent
Крок 2. Створення та активація віртуального середовища
Щоб ізолювати залежності проєкту, створіть віртуальне середовище:

Bash
python -m venv venv
Після створення його потрібно активувати:

🪟 Для Windows (PowerShell / CMD):

Bash
venv\Scripts\activate
🍎 Для macOS / Linux:

Bash
source venv/bin/activate
Крок 3. Встановлення залежностей
Встановіть усі необхідні бібліотеки:

Bash
pip install -r requirements.txt
Крок 4. Налаштування змінних середовища (.env)
Скопіюйте шаблонний файл .env.example та перейменуйте його на .env.
(Для macOS/Linux команда: cp .env.example .env)

Відкрийте файл .env та вкажіть ваші API-ключі. Файл має виглядати так:

Фрагмент кода
OPENAI_API_KEY=sk-ваш-ключ-від-openai
MODEL_NAME=gpt-4o-mini
NEWS_API_KEY=ваш-ключ-від-newsapi        # Для роботи інструменту новин
FINNHUB_API_KEY=ваш-ключ-від-finnhub     # Для роботи фондового ринку
Крок 5. Запуск помічника
Переконайтеся, що віртуальне середовище активоване, та запустіть головний файл:

Bash
python main.py
Тепер ви можете спілкуватися з агентом у консолі природною мовою (наприклад: "Яка зараз погода в Києві, курс долара, і зроби повний скан мого комп'ютера").

Для коректного завершення роботи введіть exit або натисніть Ctrl+C.
